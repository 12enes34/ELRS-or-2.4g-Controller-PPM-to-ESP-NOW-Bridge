# ELRS-or-2.4g-Controller-PPM-to-ESP-NOW-Bridge
A highly optimized, zero-latency RC car control system that reads PPM signals from a standard RC Transmitter (e.g., Jumper, Radiomaster via Trainer/AUX port) using an ESP32-S3 Super Mini, and transmits the 8-channel data wirelessly via ESP-NOW to an ESP8266 receiver equipped with an L298N motor driver.
